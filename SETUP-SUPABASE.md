# Подключение Supabase

## 1. Создать проект

Создайте новый проект в Supabase.

## 2. Выполнить SQL

Откройте `SQL Editor` в Supabase и выполните весь файл:

```text
supabase-schema.sql
```

Он создаст:

- таблицу `land_projects`;
- таблицу `land_project_files`;
- bucket `land-project-files`;
- правила доступа для вошедших сотрудников.

Если вы уже запускали этот SQL раньше, запустите его еще раз после обновлений проекта. Он добавит новые поля `address`, `status`, `responsible` и `sections`, не удаляя существующие данные.

## 3. Вставить ключи

Откройте файл:

```text
supabase-config.js
```

И замените:

```js
url: "PASTE_SUPABASE_PROJECT_URL_HERE",
anonKey: "PASTE_SUPABASE_ANON_OR_PUBLISHABLE_KEY_HERE",
```

На значения из Supabase:

- `Project URL`;
- `anon public` или `publishable` key.

## 4. Создать сотрудников

В сайте можно нажать `Создать доступ`, либо создать пользователей в Supabase Authentication.

После входа сайт будет работать в облачном режиме:

- проекты сохраняются в базе данных;
- фото и документы загружаются в Storage;
- другие вошедшие сотрудники видят общую базу;
- файлы открываются через временные защищенные ссылки.

Если ключи Supabase не вставлены, сайт работает в локальном режиме и хранит данные только в браузере.
